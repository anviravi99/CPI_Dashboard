# India CPI Analytics Dashboard (Final)

A professional, consulting-grade analytics dashboard visualizing the Consumer Price Index (CPI) of India from 2013 to 2023. This project is designed to provide clear, actionable insights into inflation trends across various economic sectors.

## Features

- **Interactive Trend Analysis**: Visualize the movement of the General Index over a decade.
- **Yearly Breakdown**: Drill down into specific years to see granular monthly data.
- **Sector Filtering**: Switch between Rural, Urban, and Combined datasets.
- **Commodity Insights**: Track price fluctuations in essential commodities like Food, Fuel, and Housing.
- **Aesthetic Design**: Clean, modern UI inspired by top-tier BI tools (Tableau/PowerBI).

## Tech Stack

- **Framework**: React 18 with TypeScript
- **Styling**: Tailwind CSS for a responsive, corporate design system
- **Charts**: Recharts for responsive, composable visualizations
- **Build Tool**: Vite

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
4. Build for production: `npm run build`

## Deployment

This project is configured for seamless deployment on platforms like Vercel or Netlify.
